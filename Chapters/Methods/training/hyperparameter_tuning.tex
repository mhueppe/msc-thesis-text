\subsection{Hyperparameter Tuning}

Hyperparameter tuning was performed using \texttt{\href{https://optuna.org/}{Optuna}} \citep{Akiba2019} to identify well-performing configurations of the neural network model. Since model performance depends strongly on the choice of training parameters (e.g., learning rate, regularization, and architectural settings), an efficient search strategy is required.

\texttt{Optuna} formulates optimization as a sequence of \emph{trials} within a \emph{study}. In each trial, a set of hyperparameters is sampled, the model is trained, and its performance on validation data is evaluated. The search space is defined programmatically, allowing different parameter types such as continuous, integer, and categorical variables to be explored flexibly.

To guide the search, \texttt{Optuna} employs adaptive sampling strategies, most notably the Tree-structured Parzen Estimator (TPE), a Bayesian optimization method. This approach uses information from previous trials to propose new configurations that are more likely to improve performance, making it more sample-efficient than grid or random search.

Additionally, \texttt{Optuna} supports \emph{pruning}, which terminates unpromising trials early based on intermediate results. This reduces computational cost and allows the optimization process to focus on more promising regions of the search space.

Model performance during tuning was evaluated on a validation set, and the optimization process iteratively refined hyperparameters to improve generalization.

In summary, \texttt{Optuna} provides a flexible and efficient framework for hyperparameter optimization, enabling systematic exploration of the search space while reducing computational overhead through adaptive sampling and early stopping.
